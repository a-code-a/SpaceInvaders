# Space Invaders Game

## Instructions

### How to Play
- Use the left and right arrow keys to move the spaceship.
- Press the spacebar to shoot projectiles.
- Destroy all enemy invaders to advance to the next level.
- Avoid enemy projectiles to stay alive.

### Objectives
- Score points by destroying enemy invaders.
- Survive as long as possible to achieve a high score.

## Basic Game Structure and Features
- Main game loop that handles game logic and rendering.
- Player-controlled spaceship that can move left and right and shoot projectiles.
- Enemy invaders that move across the screen and shoot projectiles at the player.
- Collision detection between player's projectiles and enemies, as well as between enemies' projectiles and the player.
- Scoring system to keep track of player's progress.

## Advanced Features
- Different levels with increasing difficulty.
- Power-ups that the player can collect to gain temporary advantages, such as increased fire rate or invincibility.
- Different types of enemies with unique behaviors and abilities.
- High score system to track the best scores achieved by players.
- Sound effects and background music to enhance the gaming experience.

## User Interface and Polish
- Start menu with options to start a new game, view high scores, and adjust settings.
- Pause menu that allows the player to pause the game and access options.
- Animations and visual effects to make the game more engaging.
- Responsive design to ensure the game runs smoothly on different devices and screen sizes.

## High Score System
- The high score system tracks the best scores achieved by players.
- Players can view the high scores from the start menu.
- High scores are saved and loaded automatically.
